To start the project it is necessary to create the ".env" file in the directory todo_app

.env must have variables:
TESTING_MODE
DB_USER
DB_PASS
DB_HOST
DB_PORT
DB_NAME
DB_TEST_USER
DB_TEST_PASS
DB_TEST_HOST
DB_TEST_PORT
DB_TEST_NAME
SECRET

If TESTING_MODE=True app use test arguments for database conections
